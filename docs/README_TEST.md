# tiger1.py 测试使用说明

## 快速运行测试

### 方法1: 使用清晰的测试脚本（推荐）
```bash
cd /home/cx/tigertrade
python run_test_clean.py
```

### 方法2: 使用shell脚本
```bash
cd /home/cx/tigertrade
./run_test.sh
```

### 方法3: 直接运行测试文件
```bash
cd /home/cx/tigertrade
python test_tiger1_full_coverage.py
```

## 测试输出说明

测试运行时会显示：
- ✅ 通过 - 测试成功
- ⚠️  失败 - 测试断言失败
- ❌ 错误 - 测试执行出错

每个测试会显示：
```
[编号/总数] 测试名称 ... 结果
```

## 测试文件说明

1. **test_tiger1_full_coverage.py** - 主要测试文件
   - 33个核心测试用例
   - 覆盖所有主要功能

2. **test_tiger1_additional_coverage.py** - 补充测试文件
   - 14个补充测试用例
   - 覆盖边界情况和额外路径

3. **run_test_clean.py** - 清晰的测试运行脚本
   - 过滤调试输出
   - 显示清晰的测试进度和结果

## 测试覆盖的功能

- ✅ 基础功能（指标计算、趋势判断等）
- ✅ 订单管理（下单、止盈、止损）
- ✅ 风险管理（风控检查、止损计算）
- ✅ 交易策略（网格策略、BOLL策略）
- ✅ 大模型策略（LLM训练和预测）
- ✅ 数据分析功能（市场分析、参数优化）

## 注意事项

1. 测试使用模拟API，不会进行真实交易
2. 某些测试可能需要GPU支持（大模型训练）
3. 测试输出已过滤，只显示关键信息
4. 覆盖率报告需要安装coverage模块：`pip install coverage`
