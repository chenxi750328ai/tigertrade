# 私密文档与个人信息说明

**目的**：避免将含真实 ID、账户、交易记录、账单等私密信息的文件提交到仓库；文档描述中不出现可识别的个人账号信息。

---

## 1. 不要上库的文件（已加入 .gitignore）

以下类型文件**切勿提交**，已通过 `.gitignore` 排除：

| 类型 | 示例 | 说明 |
|------|------|------|
| 账单/报表 PDF | `*账单*`、`*报表*.pdf`、`今年以来*.pdf` | 含真实账户、交易记录、ID |
| 交易原始/提取数据 | `SIL交易原始数据.txt`、`SIL交易记录.txt`、`账单报表_提取.txt` | 从账单解析，含交易明细 |
| 交易分析表格 | `SIL交易轮次分析.xlsx`、`SIL白银期货交易分析.xlsx` | 可能含真实数据 |
| API 配置 | `openapicfg_dem/tiger_openapi_config.properties`、`*.pem`、`*.key`、`.env` | 含 tiger_id、account、私钥等 |

若新增类似文件，请在 `.gitignore` 中补充规则。

---

## 2. 文档中不写真实个人信息

- **account**、**tiger_id**：文档与示例中一律用「来自配置」「&lt;配置&gt;」「勿提交」等表述，不写真实数字。
- **日志/报错示例**：用 `account 'xxx'`、`tiger_id=<配置>` 等占位，避免粘贴真实日志中的账号。
- **账单/报表**：仅说明「本地文件」「勿上库」，不写具体文件名中的个人信息。
