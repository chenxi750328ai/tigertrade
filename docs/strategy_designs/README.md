# 策略设计文档

本目录存放各交易策略的**设计文档**，每份文档均包含三块：

- **算法设计**：模型结构、输入输出、损失与推理/信号生成
- **训练过程设计**：数据划分、epoch/早停/保存、脚本入口（规则策略为回测与参数搜索）
- **数据分析设计**：特征来源、标签定义、验证集与评估、数据管道

策略报告（[reports/strategy_reports/](../reports/strategy_reports/)）中会链接到此处对应文档。
