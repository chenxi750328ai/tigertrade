# 标的与阶段说明

> 量化交易标的不限于白银期货；白银期货为**第一阶段测试标的**。

---

## 标的范围

- **本项目量化交易标的不限于白银期货**，也不限于单一品种或单一市场。
- **第一阶段**：以**白银期货**（如 COMEX SIL、SIL2603 等）为**测试标的**，用于验证数据管道、模型训练、回测与 DEMO 流程。
- **后续阶段**：可扩展至其他期货、其他标的（依 API 与策略需要），系统设计支持多标的与参数化配置。

---

## 文档中的表述约定

- 提及「白银期货」时，可加「（第一阶段测试标的）」或「当前阶段测试标的」。
- 提及「期货交易系统」时，可改为「量化交易系统」并注明标的不限于期货/白银。
- 技术实现（如 `future_contract`、期货合约格式）仍按 API 与交易所规范表述。
