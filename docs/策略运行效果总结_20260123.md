## 🔍 三、问题诊断

### 3.1 训练准确率 vs 回测收益率差异

**现象**:
- 训练准确率: 99.42%（非常高）
- 回测收益率: +2.3%（偏低）

**可能原因**:
1. **过拟合**: 模型在训练集上表现好，但泛化能力差
2. **数据泄漏**: 训练数据可能包含未来信息
3. **标签问题**: 预测准确率不等于交易盈利
4. **滑点成本**: 实际交易中的滑点侵蚀利润
5. **交易频率**: 频繁交易导致手续费成本高

### 3.2 模型复杂度问题

**发现**:
- 简单模型（53K参数）表现最好
- 复杂模型（25M+参数）表现最差

**建议**:
- 使用简单模型（LSTM）
- 避免过度复杂的模型
- 关注数据质量和特征工程

1. **时段自适应策略**（已实施）

2. **风险管理优化**
