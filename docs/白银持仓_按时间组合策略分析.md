**按走势怎么选：**

- **看涨或反弹（SLV 有机会到 115+ 甚至 120+）**  
  → **补回 120C** 更合适：再收 3,500 权利金，到期 120C 归零即多赚 3,500；若 SLV 站上约 115.38 开始转盈，站上 120 可盈约 2,309。平 100C 会放弃这部分上行空间。

- **不确定**

**一句话**：**看涨/看反弹至 115+ → 补回 120C；看跌或横盘、想止损 → 平 100C。**
